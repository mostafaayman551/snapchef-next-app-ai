# SnapChef

SnapChef is a web application that uses AI to generate recipes from a single photo of a dish. The application is built using Next.js, Tailwind CSS, and Prisma.

## Features

- Take a photo of a dish and get a recipe generated by AI
- Search for recipes by ingredient or dish name
- Save and organize recipes in a personal cookbook
- Delete recipes

## Packages Used

- Next.js
- Tailwind CSS
- Prisma
- Lucide (icons)
- React Redux
- Redux Toolkit
- Zod
- Axios
- bcrypt
- jsonwebtoken
- redux-persist

## Steps to Use

1. Clone the repository and install the dependencies with `npm install` or `yarn install`.
2. Start the development server with `npm run dev` or `yarn dev`.
3. Open the application in your browser at `http://localhost:3000`.
4. Take a photo of a dish and upload it to the application.
5. The application will generate a recipe based on the photo.
6. Search for recipes by ingredient or dish name.
7. Save and organize recipes in your personal cookbook.
8. Edit and update recipes.
9. Delete recipes.
